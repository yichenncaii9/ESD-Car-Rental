<objective>
Create Kubernetes manifests for the 4 wrapper services (openai, googlemaps, stripe, twilio-http) and websocket-server. This plan runs in parallel with Plan 03 (atomic services) since the file sets are completely independent.

Purpose: Wrappers are called by composite services; websocket-server is called by workers. Both groups must be in the cluster before composite/worker manifests can resolve them.
Output: 15 manifest files covering 5 services (websocket-server now has ConfigMap for K8S-01 compliance)

Note on workers: twilio-worker and activity-log deliberately have no Service manifest — they are pure AMQP consumers with no inbound HTTP traffic. K8S-01 "Deployment + Service + ConfigMap" applies to HTTP-serving pods only; workers are the intentional exception.
</objective>

**openai-wrapper (port 6200, key: OPENAI_API_KEY):**

k8s/openai-wrapper/configmap.yaml:
```yaml
apiVersion: v1
kind: ConfigMap
metadata:
  name: openai-wrapper-config
  namespace: default
data:
  PORT: "6200"
```

k8s/openai-wrapper/deployment.yaml:
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: openai-wrapper
  namespace: default
spec:
  replicas: 1
  selector:
    matchLabels:
      app: openai-wrapper
  template:
    metadata:
      labels:
        app: openai-wrapper
    spec:
      containers:
        - name: openai-wrapper
          image: esd-openai-wrapper:latest
          imagePullPolicy: Never
          ports:
            - containerPort: 6200
          env:
            - name: OPENAI_API_KEY
              valueFrom:
                secretKeyRef:
                  name: api-keys
                  key: OPENAI_API_KEY
          livenessProbe:
            httpGet:
              path: /health
              port: 6200
            initialDelaySeconds: 20
            periodSeconds: 15
            failureThreshold: 5
          readinessProbe:
            httpGet:
              path: /health
              port: 6200
            initialDelaySeconds: 20
            periodSeconds: 15
            failureThreshold: 5
```

k8s/openai-wrapper/service.yaml:
```yaml
apiVersion: v1
kind: Service
metadata:
  name: openai-wrapper
  namespace: default
spec:
  selector:
    app: openai-wrapper
  ports:
    - port: 6200
      targetPort: 6200
  type: ClusterIP
```
